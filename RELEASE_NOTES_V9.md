# 反诈人生 V9.0 更新说明

> 一个用**沉浸式叙事**帮 Z 世代识别和防范电信网络诈骗的 Web 应用。
> 在 V8「完整五幕剧情」的基础上，V9 补上了**安全护栏、实战反诈内容、游戏化机制、复玩动力**，并让它能**装进手机 / 微信小程序、能听、能量化效果**。

---

## 一句话：V9 比 V8 多了什么？

V8 让你「体验一遍被骗的过程」；V9 让你「**真正学会在现实里识破骗局**」——
既能安全地把十大高发诈骗逐一练熟，又能看清自己最容易在哪类话术上中招，还能在手机、小程序里随手就玩。

---

## 新增 / 升级能力（面向玩家）

### 🛡️ 安全护栏（底层守护）
- 全程拦截「套取联系方式、诱导转账话术、越狱提示词」等危险输入/输出，绝不把真实诈骗手法或个人信息泄漏出去。
- 即使玩家故意「试探」，也只会得到安全的兜底回应，安全合规。

### 🎯 十大诈骗类型 · 实战演练
覆盖当前最高发、最贴近年轻人的骗局，每种都配 **Z 世代话术样本 + 同龄人被骗真实案例 + 防骗口诀**：
1. 刷单返利　2. 冒充电商客服　3. 杀猪盘（情感交友）　4. 虚假投资理财
5. 冒充公检法　6. 游戏账号交易　7. 校园贷 / 注销校园贷　8. 兼职免费送
9. AI 换脸 / 拟声　10. 冒充熟人 / 领导

每个场景含 **3 种话术变体 + 4 枚证据碎片 + 复盘小结**，玩一遍就把红标（踩坑点）记牢。

### 🧠 心理弱点系统
系统会识别你最容易中招的心理点：**孤独 / 贪婪 / 虚荣 / 恐惧 / 急躁**，
并在关键节点弹出「内心独白」，帮你看见「为什么我会心动」——识破骗局的第一步，是识破自己。

### 🧩 证据墙 · 反杀骗子
在剧情里收集证据碎片，凑齐后可触发「反杀」结局，从受害者变成破局者。

### ⚡ 快速模式（Reigns 式）
不想读长剧情？左右滑卡即可做抉择，四维数值即时反馈（信任 / 暴露 / 怀疑 / 良心），
一局 3–5 分钟，适合碎片时间刷反诈直觉。

### 🦋 蝴蝶效应 · 多结局图鉴
你的每个选择都会改变走向；全部结局收入**图鉴**，二周目收集解锁，越玩越懂骗子套路。

### 🔊 语音播报
「骗子 / AI」消息支持**中文语音朗读**（浏览器内置，零流量），通勤路上也能「听」骗局。

### 📱 小程序 / H5 适配
自动识别微信 / 小程序 `web-view` 环境，适配刘海屏与安全区；
在小程序里放一个 `<web-view>` 就能把游戏装进去，手机、Pad 都能玩。

### 📊 匿名数据埋点 + 离线看板（给运营 / 创作者）
- 只采集**匿名聚合事件**（开始游戏、踩中红标、看到报告等），**不记录姓名、手机号、聊天原文**。
- 一键生成离线看板 `dashboard.html`，看清「哪些骗局最容易被踩、哪个环节掉落最多」，持续优化教育内容。

## 隐私与安全承诺
- **全程不索取任何真实个人信息**（姓名、电话、账号、地址等）。
- 埋点**仅做匿名计数**，不含对话原文；本地无后端时自动降级，不影响游玩。
- 内容含防骗教育提示与「一键退出」，任何不适可随时离开。

---

## 怎么玩
1. 网页直接打开 `index8.html`（无需安装、无需后端）。
2. 选 / 随机一个身份，进入五幕剧情：希望 → 崩塌 → 废墟 → 回放 → 盾牌。
3. 想专项练某种骗局 → 进「场景实战」；想快速刷直觉 → 进「快速模式」；想听 → 开 🔊 语音。
4. 通关后看「盾牌报告 + 避坑报告」，把易受骗点记下来。
