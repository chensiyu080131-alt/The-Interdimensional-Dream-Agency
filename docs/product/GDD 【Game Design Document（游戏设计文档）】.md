| 项目        | 内容                                                                                                              |
| --------- | --------------------------------------------------------------------------------------------------------------- |
| **项目名称**  | 《反诈人生》（MVP版）                                                                                                    |
| **核心玩法**  | 玩家在仿微信聊天界面中，与AI生成的“骗子”对话。聊天过程中骗子逐渐诱导转账，玩家转钱后骗子消失，随即弹出财务崩塌模拟。                                                    |
| **目标平台**  | Web端（移动端优先），评委扫码即玩                                                                                              |
| **MVP范围** | **第一幕「邂逅」** ：AI生成骗子人设与话术，玩家自由聊天，体验“养猪”过程  <br>**第二幕「崩塌」前半段**：玩家转账后骗子消失，弹出财务模拟面板（房租、账单、余额），展示“钱没了”的即时后果          |
| **AI使用点** | ①混元大模型生成骗子话术（3种人设随机：投行精英/军人/医生）  <br>②AI生成玩家“财务档案”（基于输入的年收入、月支出）                                                |
| **技术栈**   | 前端：HTML + CSS + JavaScript（单页应用）  <br>后端：Node.js + Express（API层）  <br>AI：腾讯混元大模型API  <br>部署：腾讯云CloudBase        |
| **开发工具**  | CodeBuddy（全程）                                                                                                   |
| **美术风格**  | 仿微信聊天界面。骗子出现阶段：暖色调（橙色/米白）；转账后：冷灰色调，营造“崩塌”感                                                                      |
| **开发周期**  | 7月14日–7月19日（6天）                                                                                                 |
| **验收标准**  | ①打开页面看到聊天界面  <br>②骗子主动发消息，AI实时回复  <br>③玩家可自由输入回复  <br>④骗子在适当节点诱导转账  <br>⑤玩家点击转账后骗子消失+财务面板弹出  <br>⑥所有内容在手机上可正常浏览 |
